## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/bbq-page.git
```

Go to the project directory

```bash
  cd ..bbq-page
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run start
```
